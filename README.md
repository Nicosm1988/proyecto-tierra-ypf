# Proyecto Tierra YPF

Globo terraqueo 3D interactivo con estilo de radar ejecutivo para visualizar ubicaciones, flujos y novedades de operaciones energeticas.

## Stack

- Vite + React
- globe.gl sobre Three.js
- Datos locales en `src/data`
- Texturas estaticas en `public/assets`

## Desarrollo

```bash
npm install
npm run dev
```

## Produccion

```bash
npm run build
```

Vercel detecta la app como Vite y publica el contenido generado en `dist`.
